# Co-Agent-v1 — Console AI Agent

A fast, minimal console AI agent that can search the web and manage files — all from your terminal.

## Requirements
- Python 3.10+
- Groq API key → https://console.groq.com
- Tavily API key → https://app.tavily.com

## Installation
git clone https://github.com/gygyLonely/Co-Agent-V1
cd Co-Agent-v1
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
nano .env

## Usage
python3 Co-Agent-V1.py
